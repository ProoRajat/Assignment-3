# Assignment-3
In this assignment, a Tic-Tac-Toe game was created using the C programming language.

The game runs in the terminal.

Players can compete against a computer opponent.

The board updates dynamically after each move, and win/draw conditions are checked automatically.

## Compile and Run
Compiling
Open the terminal in the project directory and run:
``` bash
gcc tictactoe.c -o tictactoe
```
Running
Execute the compiled programs with:
```bash
./tictactoe
```
